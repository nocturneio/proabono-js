### :cloud: Installation

**Only one step :**
```shell
npm install --save proabono-js
```
or
```shell
yarn add proabono-js
```

```js
let pa = new ProAbono('AGENT-KEY', 'API-SECRET', 'ENDPOINT');
pa.customers().list(); -> return Promise
```
